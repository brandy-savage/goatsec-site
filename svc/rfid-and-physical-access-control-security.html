RFID and Physical Access Control Security

Assessment of RFID-based physical access control systems, card cloning feasibility, reader attack surface, credential brute-force windows, and upgrade paths for legacy systems.

Best for

Any organization using card-based access control, especially those still running legacy 125 kHz proximity cards (HID Prox, EM4100) that are trivially cloneable.

What we deliver
  • Card technology inventory with cloning feasibility ratings
  • Reader configuration security findings
  • Credential management gap analysis
  • Recommended upgrade path (125 kHz to MIFARE or SEOS)
  • Physical access log review findings
How we work
  • Passive card interrogation and technology identification
  • Clone feasibility test with Proxmark or Flipper Zero
  • Anti-replay and challenge-response evaluation
  • Reader-to-controller communication review
  • Physical lock and door gap assessment
